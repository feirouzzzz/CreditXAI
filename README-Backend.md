# How backend works

## Authentification
- user signup (email, full_name, password)
- verify identity (CIN)

- user login (email, password)


### Logged in
#### Complete Docs
- Income & Employment Verification
    - Recent pay slips (last 3 months) (PDF)
    - Tax declaration (PDF)

- Financial Activity
    - Income consistency (3 months) (CSV)

- Loan History Documents
    - Loan payments (CSV)

- Business Documents (If the user is self-employed)    
    For freelancers, entrepreneurs, or business owners.
    - Business registration certificate (PDF)
    - Income declaration (PDF)
